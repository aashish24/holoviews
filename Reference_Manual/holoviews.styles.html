holoviews.styles Package¶

styles Package¶

Defines global matplotlib styles and HoloViews-specific style options.

For setting a style, see set_style.

holoviews.styles.set_style(key)[source]¶

Select a style by name, e.g. set_style(‘default’). To revert to the previous style use the key ‘unset’ or False.
